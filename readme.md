# Laravel Mix Without Laravel

[Laravel Mix docs](https://github.com/JeffreyWay/laravel-mix/tree/master/docs#readme)

## Installation

To install all dependencies:

```
npm install
```

## Usage

Run all tasks:

```
npm run dev
```

Run all tasks and minify:

```
npm run production
```

Run all tasks and watch for changes:

```
npm run watch
```
